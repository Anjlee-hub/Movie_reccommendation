# 🎬 Movie Recommendation System

An AI-powered movie recommendation web application built using Python, NLP, Machine Learning, and Streamlit.

## Features
- Content-based recommendation system
- Movie poster integration using TMDB API
- Smart fuzzy movie search
- Cinematic UI with animations
- Streamlit interactive frontend

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit
- NLTK
- TMDB API

## ML Concepts Used
- NLP preprocessing
- Stemming
- CountVectorizer
- Cosine Similarity
- Feature Engineering

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app/app.py
